# Ancestral Ocean Intelligence

Proyecto abierto que explora la relación entre naturaleza y bienestar humano.

La app registra:

- energía
- ánimo
- estrés
- sueño
- tiempo en naturaleza
- surf
- pesca
- cannabis
- alimentación
- reflexiones personales

El objetivo es generar **conocimiento abierto** sobre cómo los ritmos naturales afectan la salud humana.

## Ejecutar

```bash
pip install -r requirements.txt
streamlit run app.py
